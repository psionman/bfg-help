    Menu¶

    To the right of the screen a menu will appear. It may be revealed and hidden using the ‘hamburger’ button in the top right of the main screen. A menu item might be disabled (dimmed) if it cannot function at that stage.

    Boards¶

    New board¶

    Deals a new board (from Set hands if selected.)

    Board selection¶

    Takes you to the History screen.

    Set hands¶

    Takes you to the Set hands screen.

    Control¶

    Restart board¶

    Removes all actions and presents the board in its original form.

    Replay board¶

    Return the board to the start of card play. The bidding and contract remain intact.

    Undo¶

    Undoes the last action.

    Claim/Concede¶

    Displays the Claim/concede dialog.

    Display Auction¶

    At any time during cardplay you may display the auction for the board.

    Previous trick¶

    Displays the previous trick played.

    Makeable tricks¶

    Disp;lays the number of makebale tricks in each denimination for each play on this board.

    Application¶

    Mode¶

    Returns to the Mode screen (the current hand will be lost but may be retrieved from History).

    Preferences¶

    Display the preferences screen.

    Help¶

    Display on-line user documentation.

    Refresh¶

    If your screen has become corrupted or does not operate when expected, then Refresh might help resolve the problem.

    Logout¶

    Logs you out of the application.
