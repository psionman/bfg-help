# Set Hands

Set hands allow you to bid based on specific criteria; for example, *Responding to 2C opening bid*. You will see a different lst of *Set hand* options depending on whether you are playing in the Solo or Duo mode. In Duo mode, both you and your partner will see the same *Set hands* .

To access set hands, click on *Set hands*  in the [Menu](menu). You will see a screen similar to

![Random hands](/_images/set_hands01.png)

To see the set hands option click on the *Set hands* radio button.

You will then be able to select one or more set hand options. If you choose more than one, then the hands will be generated randomly from the list. Click on *Submit* to accept.

![Select set hands](/_images/set_hands02.png)

The [Board source display](main_screen.html#board-source) will confirm that you are using set hands.

![Set hands source](/_images/board_source01.png)

The *Display hand type* checkbox will appear below *Set hands* button in the [Menu](menu).

![Display hand type](/_images/display_hand_type01.png)

If it is checked the set hand description will appear in the [Board source display](main_screen.html#board-source)

![Set hands source with hand type](/_images/board_source02.png)

To remove this information, uncheck the *Display hand type* checkbox.
